# BiliPai (个人定制自用版)

本项目是基于第三方原生 Bilibili Android 客户端 [BiliPai](https://github.com/jay3-yy/BiliPai) 的个人定制版本。

## 定制功能

- **去除展开评论面板及二级评论时的灰色遮罩**：
  - 在官方原版中，展开评论区面板或进入二级评论（楼中楼）详情页时，视频区域会被一层半透明的灰色遮罩覆盖。
  - 由于展开二级评论和浏览评论是极高频的操作，频繁出现灰色遮罩非常影响视频观看的连贯体验。
  - 本定制版彻底移除了这层灰色遮罩（将遮罩透明度设为 `0f`），让您在频繁展开/关闭评论与楼中楼时，能够获得更加无缝、纯净的观看与浏览体验。
  - **发表评论（输入框弹出）时的灰色遮罩保持官方原版设计**。
